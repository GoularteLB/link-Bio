## Sobre o projeto

O **link‑Bio** é um template simples em formato mobile, ideal para usar em biografias de redes sociais como Instagram.  
O objetivo é reunir seus principais links num único lugar, com visual limpo e responsivo, pensado para celular, otimizado para UI/UX e desenvolvido com Vue.js e Tailwind CSS.

## Motivação

- **Aprendizado prático**: aprimorar habilidades em Tailwind CSS e desenvolvimento com Vue.js.  
- **Portfólio funcional**: criar uma base visual atrativa que reúna suas principais redes e contatos em um layout moderno.  
- **Foco mobile**: como as pessoas acessam redes sociais majoritariamente pelo celular, o design prioriza uma experiência móvel responsiva.  
- **Uso livre como exemplo**: o projeto está disponível publicamente para servir de inspiração ou base para outros desenvolvedores.

## Stack

- **Framework**: Vue.js  
- **Estilização**: Tailwind CSS  
- **Design UI/UX**: inspirado em layouts mobile típicos de redes sociais, idealizado no Figma com foco na usabilidade e estética.
  

## Principais características

- Responsivo e otimizado para mobile  
- Design clean e baseado em conceitos de UI/UX  
- Fácil personalização de links e conteúdo  
- Ideal para portfólio e uso em redes sociais  
- Código aberto — pode ser usado como exemplo
